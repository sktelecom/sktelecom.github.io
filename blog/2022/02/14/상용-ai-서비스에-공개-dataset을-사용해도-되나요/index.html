Can I use this publicly available dataset to build commercial AI software?
By мһҘн•ҷм„ұ | 2022л…„ 2мӣ” 14мқј

м•Ҳл…•н•ҳм„ёмҡ”, мһҘн•ҷм„ұмһ…лӢҲлӢӨ.

AIлҠ” мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” кё°м—…мқҙ м—Ҷмқ„ м •лҸ„лЎң нҳ„лҢҖ л№„мҰҲлӢҲмҠӨм—җ мӨ‘мҡ”н•ң кё°мҲ мқҙ лҗҳм—ҲмҠөлӢҲлӢӨ. AI м„ңл№„мҠӨлҘј л§Ңл“Өкё° мң„н•ҙм„ңлҠ” л§ҺмқҖ м–‘мқҳ dataк°Җ н•„мҡ”н•ңлҚ°мҡ”, кіөк°ң Datasetpublicly available datasetsлҸ„ л„җлҰ¬ мӮ¬мҡ©лҗҳкі  мһҲмҠөлӢҲлӢӨ. лӢӨл§Ң кіөк°ң Datasetмқҙлқјкі  н•ҳлҚ”лқјлҸ„ м Җмһ‘к¶Ңмқҙ мһҲкё° л•Ңл¬ём—җ мқҙлҘј мғҒмҡ© AI м„ңл№„мҠӨм—җ мӮ¬мҡ©н•ҳл Өл©ҙ м Җмһ‘к¶Ң м№Ён•ҙ л“ұ лІ•м Ғ лҰ¬мҠӨнҒ¬лҘј мөңмҶҢнҷ”н•ҳкё° мң„н•ҙ лқјмқҙм„ мҠӨ мёЎл©ҙмқҳ нҷ•мқёмқҙ н•„мҡ”н•©лӢҲлӢӨ.

мҳӨлҠҳмқҖ мқҙмҷҖ кҙҖл Ён•ҳм—¬ мөңк·ј л°ңн‘ңлҗң л…јл¬ёмқё Can I use this publicly available dataset to build commercial AI software?– A Case Study on Publicly Available Image Datasetsмқ„ мҶҢк°ңн•ҳл Өкі  н•©лӢҲлӢӨ. : https://arxiv.org/abs/2111.02374

“Can I use this publicly available dataset to build commercial AI software? – A Case Study on Publicly Available Image Datasets”

- Gopi Krishnan Rajbahadur, Erika Tuck, Li Zi, Dayi Lin, Boyuan Chen, Zhen Ming (Jack)Jiang, Daniel Morales German

мқҙ кёҖмқ„ нҶөн•ҙ кіөк°ң Datasetмқ„ нҷңмҡ©н•ң AI м„ңл№„мҠӨлҘј мӨҖл№„н•ҳл©ҙм„ң м Җмһ‘к¶Ң м№Ён•ҙлҘј мөңмҶҢнҷ”н•ҳкё° мң„н•ҙ м–ҙл–Ө л…ёл Ҙкіј м Ҳм°ЁлҘј кұ°міҗм•ј н•ҳлҠ”м§Җм—җ лҢҖн•ң мқёмӮ¬мқҙнҠёлҘј м–»мқ„ мҲҳ мһҲкё°лҘј л°”лһҚлӢҲлӢӨ.

1. Intro

мқҙ л…јл¬ём—җм„ңлҠ” лЁјм Җ кіөк°ң Datasetмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ң лқјмқҙм„ мҠӨлҠ” мҳӨн”ҲмҶҢмҠӨ лқјмқҙм„ мҠӨмҷҖлҠ” лӢ¬лҰ¬ лӘҮ к°Җм§Җ м–ҙл Өмҡҙ л¬ём ңк°Җ мһҲлӢӨкі  м„ӨлӘ…н•©лӢҲлӢӨ.

1. н•ҙлӢ№ Datasetм—җ лҢҖн•ң мҷ„м „н•ҳкі  м •нҷ•н•ң лқјмқҙм„ мҠӨлҘј мӢқлі„н•ҳкё°к°Җ м–ҙл өлӢӨ. - мҳҲлҘј л“Өм–ҙ, Datasetмқ„ м ңкіөн•ҳлҚҳ мӣ№мӮ¬мқҙнҠёк°Җ нҸҗмҮ„лҗҳлҠ” кІҪмҡ°лҸ„ мһҲлӢӨ. 2. н•ҙлӢ№ Datasetм—җ лҢҖн•ң лқјмқҙм„ мҠӨк°Җ мң нҡЁн•ңм§Җ нҷ•мқён•ҳлҠ” кІғмқҙ м–ҙл өлӢӨ. - л§ҺмқҖ DatasetмқҖ м—¬лҹ¬ к°ңмқҳ Data SourceлҘј кІ°н•©н•ҳм—¬ мғқм„ұн•ңлӢӨ. мқҙлҹ¬н•ң м—¬лҹ¬ Data SourceлҠ” к°Ғк°Ғ лӢӨлҘё лқјмқҙм„ мҠӨк°Җ м Ғмҡ©лҗңлӢӨ. - лҚ”кө¬лӮҳ publicly available datasetмқҳ мһ‘м„ұмһҗлҠ” Datasetмқ„ л§Ңл“Өл©ҙм„ң мӮ¬мҡ©н•ң лӢӨм–‘н•ң Data Sourceмқҳ лқјмқҙм„ мҠӨлҘј кұ°мқҳ л¬ём„ңнҷ”н•ҳм§Җ м•ҠлҠ”лӢӨ. - мҳҲлҘј л“Өм–ҙ, [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html) DatasetлҸ„ мӣ№мӮ¬мқҙнҠём—җм„ң мҡ”кө¬н•ҳлҠ” мң мқјн•ң лқјмқҙм„ мҠӨлҠ” вҖҳмқёмҡ© мҡ”кө¬' лҝҗмқҙкі  к·ё мҷём—җлҠ” м„ӨлӘ…н•ҳм§Җ м•ҠлҠ”лӢӨ. - н•ҳм§Җл§Ң, [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html)лҠ” мғҒм—…м Ғ мқҙлҜём§Җ мӮ¬мҡ©мқ„ м ңн•ңн•  мҲҳ мһҲлҠ” лқјмқҙм„ мҠӨк°Җ м Ғмҡ©лҗң Google Images, FlickrмҷҖ к°ҷмқҖ лӢӨм–‘н•ң Data Sourceмқҳ мқҙлҜём§ҖлҘј нҒ¬лЎӨл§Ғн•ҳм—¬ мғқм„ұлҗҳм—ҲлӢӨ. - мқҙлҹ¬н•ң кІҪмҡ°, [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html)мқҳ лқјмқҙм„ мҠӨл§Ңмқ„ кі л Өн•ҳлҠ” кІғмқҖ м»ҙн”Ңлқјмқҙм–ёмҠӨ л¬ём ңк°Җ лҗ  мҲҳ мһҲлӢӨ. 3. Publicly available datasetм—җ м Ғмҡ©лҗң лқјмқҙм„ мҠӨлҠ” мқјл°ҳм ҒмңјлЎң лӘЁнҳён•ҳм—¬ к¶ҢлҰ¬мҷҖ мқҳл¬ҙлҘј лӘ…нҷ•н•ҳкІҢ м„ӨлӘ…н•ҳм§Җ м•ҠлҠ”лӢӨ. - мқҙлҹ¬н•ң Datasetмқ„ мӮ¬мҡ©н•ҳм—¬ лқјмқҙм„ мҠӨ лҰ¬мҠӨнҒ¬ м—Ҷмқҙ мғҒмҡ© AI softwareлҘј кө¬м¶•н•ҳлҠ” кІғмқҖ мӢӨм ңлЎң м–ҙл өлӢӨ. - мҳҲлҘј л“Өм–ҙ, GitHub CopilotмқҖ Githubм—җм„ң нҳёмҠӨнҠёлҗҳлҠ” мҲҳмӢӯм–ө мӨ„мқҳ source codeлЎң trainingн•ң лҢҖк·ңлӘЁ AI Modelмқ„ мӮ¬мҡ©н•ңлӢӨ. - к·ёлҹ¬лӮҳ мҳӨн”ҲмҶҢмҠӨ лқјмқҙм„ мҠӨм—җм„ңлҠ” мҶҢмҠӨ мҪ”л“ңлҘј мғҒм—…м Ғ лӘ©м ҒмңјлЎң AI Modelмқ„ training н•ҳлҠ” лҚ° мӮ¬мҡ©н•  к¶ҢлҰ¬м—җ лҢҖн•ҙ лӘ…нҷ•н•ҳкІҢ м •мқҳлҗҳм§Җ м•Ҡм•ҳлӢӨ. - мқҙлҹ¬н•ң лӘЁнҳём„ұмқҖ GitHub Copilotмқҳ м»ҙн”Ңлқјмқҙм–ёмҠӨм—җ лҢҖн•ң кҙ‘лІ”мң„н•ң лІ•м Ғ л…јмҹҒмңјлЎң мқҙм–ҙмЎҢлӢӨ.

GitHub Copilot

м—¬кё°м„ң мһ к№җ GitHub Copilotкіј кҙҖл Ён•ң л…јмҹҒм—җ лҢҖн•ҙ м–ёкёүн•ҳкі  л„ҳм–ҙк°ҖкІ мҠөлӢҲлӢӨ. мөңк·ј лҜёкөӯмқҳ SFCSoftware Freedom Conversancyм—җм„ңлҠ” “If Software is My Copilot, Who Programmed My Software?“лқјлҠ” кёҖмқ„ кІҢмһ¬н•ҳм—¬ MicrosoftмҷҖ GitHubмқҳ мЈјмһҘмқ„ л°ҳл°•н•ҳмҳҖмҠөлӢҲлӢӨ.

1. ML systemмқ„ training н•ҳкё° мң„н•ң public dataмқҳ мӮ¬мҡ©мқҖ Fair UseмқҙлӢӨ. 2. ML systemмқҙ л§Ңл“  outputмқҖ мӢңмҠӨн…ң operatorм—җкІҢ мҶҚн•ңлӢӨthe output belongs to the operator.

н•ҳм§Җл§Ң, SFCлҠ” мқҙлҹ¬н•ң GitHubмқҳ мһ…мһҘмқҖ Copilot лӢӨмқҢкіј к°ҷмқҙ мӮ¬мҡ©мһҗм—җкІҢ нҒ° н”јн•ҙлҘј мӨ„ мҲҳ мһҲлӢӨкі  кІҪкі н•ҳмҳҖмҠөлӢҲлӢӨ. л”°лқјм„ң лӢӨлҘё мӮ¬лһҢмқҳ м Җмһ‘к¶Ңмқ„ м№Ён•ҙн•ҳм§Җ м•Ҡмңјл Өл©ҙ Copilotмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” кІғмқҙ мўӢлӢӨлҠ” мһ…мһҘмқ„ н‘ңлӘ…н•ҳмҳҖмҠөлӢҲлӢӨ.

- вҖңthe output belongs to the operatorвҖқлқјлҠ” GitHubмқҳ мЈјмһҘмқҖ мһҳлӘ»лҗң лІ•м Ғ м •лӢ№м„ұмқ„ л§Ңл“ лӢӨ. - GitHub CEOмқҳ м§„мҲ мқҖ GPL enforcement actionм—җ м§Ғл©ҙн•  мҲҳ мһҲлҠ” Copilot мӮ¬мҡ©мһҗм—җ лҢҖн•ң л°°мғҒ мұ…мһ„мқ„ нҡҢн”јн•ңлӢӨ. - кІ°көӯ мӮ¬мҡ©мһҗк°Җ Copilotмқҳ outputм—җ лҢҖн•ң вҖҳFair UseвҖҷ лҳҗлҠ” вҖҳnot copyrightableвҖҷн•ҳлӢӨлҠ” л°©м–ҙмұ…мқ„ л§Ҳл Ён•ҙм•ј н•ңлӢӨлҠ” л§җмқҙлӢӨ.

2. Background

кі„м•ҪлІ•
кі„м•ҪлІ•м—җ л”°лҘҙл©ҙ м Җмһ‘л¬ј(мҳҲ: мқҙлҜём§Җ, л№„л””мҳӨ)мқҳ м Җмһ‘к¶Ң мҶҢмң мһҗлҠ” лӢӨлҘё мӮ¬лһҢмқҙ н–Ҙмң н•  мҲҳ мһҲлҠ” к¶ҢлҰ¬мҷҖ к·ёлҹ¬н•ң к¶ҢлҰ¬лҘј н–Ҙмң н•ҳкё° мң„н•ҙ мқҙн–үн•ҙм•ј н•ҳлҠ” мқҳл¬ҙлҘј м„ӨлӘ…н•ҳлҠ” лқјмқҙм„ мҠӨлҘј л¶Җм—¬н•  мҲҳ мһҲлӢӨ. - лқјмқҙм„ мҠӨ мЎ°кұҙмқҙ мЎҙмӨ‘лҗҳм§Җ м•ҠлҠ” кІҪмҡ°, мҰү лқјмқҙм„ мҠӨк°Җ л¶Җм—¬н•ҳм§Җ м•ҠмқҖ к¶ҢлҰ¬лҘј dataк°Җ н–үмӮ¬н•ҳкұ°лӮҳ мқҳл¬ҙк°Җ мқҙн–үлҗҳм§Җ м•ҠлҠ” кІҪмҡ° (мһ мһ¬м Ғ) кі„м•Ҫ мң„л°ҳ лҳҗлҠ” кі„м•Ҫ мң„л°ҳм—җ н•ҙлӢ№н•  мҲҳ мһҲлӢӨ.

кІ°көӯ кіөк°ң Datasetмқ„ мӮ¬мҡ©н•ҳм—¬ AI м„ңл№„мҠӨлҘј к°ңл°ңн•ҳлҠ” кё°м—…мқҖ (Fair UseлЎң нҢҗлӢЁн•  мҲҳ мһҲлҠ” кІҪмҡ°лҘј м ңмҷён•ңлӢӨл©ҙ) м Җмһ‘к¶Ңм№Ён•ҙ, кі„м•ҪлІ• мң„л°ҳ л“ұмқ„ л°©м§Җн•ҳкё° мң„н•ҳм—¬ кіөк°ң Datasetкіј кҙҖл Ёлҗң к¶ҢлҰ¬мҷҖ мқҳл¬ҙлҘј нҷ•мқён•ҳкі  лқјмқҙм„ мҠӨ м»ҙн”Ңлқјмқҙм–ёмҠӨлҘј ліҙмһҘн•ҳкё° мң„н•ң м—„кІ©н•ң м ‘к·ј л°©мӢқмқҙ мӨ‘мҡ”н•ҳлӢӨкі  к°•мЎ°н•©лӢҲлӢӨ.

к·ёлҹ°лҚ° мқҙнӣ„м—җ лӢӨмӢң м–ёкёүн•ҳкІ м§Җл§Ң мӮ¬мӢӨ кіөк°ң Datasetмқ„ мӮ¬мҡ©н•ҳл©ҙм„ң Dataset, Data Source лҝҗл§Ң м•„лӢҲлқј data point л“ұмқҳ лӘЁл“  лқјмқҙм„ мҠӨлҘј нҷ•мқён•ҳкі  к°Ғк°Ғмқҳ мқҳл¬ҙлҘј мӨҖмҲҳн•ҳлҠ” кІғмқҖ кұ°мқҳ л¶Ҳк°ҖлҠҘм—җ к°Җк№қмҠөлӢҲлӢӨ. кіөк°ң Datasetмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ҙ мқјм • л¶Җл¶„мқҳ лқјмқҙм„ мҠӨ лҰ¬мҠӨнҒ¬лҘј к°җмҲҳн•ҳкұ°лӮҳ Fair Useлқјкі  мЈјмһҘн•  мҲҳ мһҲлҠ” лІ•м Ғ к·јкұ°лҘј л§Ҳл Ён•ҳлҠ” кІғмқҙ нҳ„мӢӨм Ғмқё лҢҖмқ‘ л°©м•Ҳмқҙлқјкі  к°ңмқём ҒмңјлЎң мғқк°Ғн•©лӢҲлӢӨ.

к·ёлҹј мқҙ л…јл¬ём—җм„ң м ңм•Ҳн•ҳлҠ” кіөк°ң Datasetмқ„ мғҒмҡ© AI м„ңл№„мҠӨм—җ нҷңмҡ©н•ҳкё° мң„н•ң м—„кІ©н•ң м ‘к·ј л°©мӢқмқҙ л¬ҙм—Үмқём§Җ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

### External validity мқҙ л…јл¬ём—җм„ңлҠ” лқјмқҙм„ мҠӨ мң„л°ҳ мёЎл©ҙл§Ң м—°кө¬н•ҳмҳҖлӢӨ. - Datasetмқ„ мӮ¬мҡ©н•ҳм—¬ AI SoftwaeлҘј кө¬м¶•н•  л•Ңм—җлҠ” к°ңмқё м •ліҙ ліҙнҳё, мңӨлҰ¬м Ғ л¬ём ңмҷҖ к°ҷмқҖ мҡ”мҶҢлҸ„ мӨ‘мҡ”н•ҳлӢӨ. - Datasetмқ„ лӮҙл¶Җ м—°кө¬, н•ҷл¬ём Ғ лӘ©м ҒмңјлЎң мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°лҸ„ лӢӨлЈЁм§Җ м•Ҡм•ҳлӢӨ. - Fair Use, fair dealing, кё°нғҖ мң мӮ¬н•ң лІ•лҘ м—җ л”°лқј мӮ¬мҡ©н•  мҲҳ мһҲлҠ”м§ҖлҸ„ лӢӨлЈЁм§Җ м•Ҡм•ҳлӢӨ. - мқҙ м—°кө¬м—җм„ңлҠ” мқҙлҜём§Җ Datasetл§Ң лӢӨлЈЁм—ҲлӢӨ. video, text л“ұ лӢӨлҘё Datasetм—җм„ңлҠ” лҳҗ лӢӨлҘё л¬ём ңк°Җ м ңкё°лҗ  мҲҳ мһҲкё°м—җ м¶”к°Җ м—°кө¬к°Җ н•„мҡ”н•ҳлӢӨ. ### Internal validity мқҙ м—°кө¬м—җм„ңлҠ” Data Sourceмқҳ лқјмқҙм„ мҠӨк№Ңм§Җл§Ң кі л Өн•ҳкі , к°ңлі„ data point (мҳҲ: к°ңлі„ мқҙлҜём§Җ)мҷҖ кҙҖл Ёлҗң лқјмқҙм„ мҠӨлҠ” кі л Өн•ҳм§Җ м•Ҡм•ҳлӢӨ. - к°ңлі„ мқҙлҜём§Җм—җлҸ„ м Җмһ‘к¶Ңмқҙ мһҲмқ„ мҲҳ мһҲлӢӨ. - н•ҳм§Җл§Ң, к°Ғ Data Sourceм—җлҠ” мҲҳмІң, мҲҳл§Ң к°ңмқҳ data pointк°Җ мһҲлҠ”лҚ°, мқҙм—җ лҢҖн•ң к°Ғ лқјмқҙм„ мҠӨлҘј м¶”м¶ңн•ҳлҠ” кІғмқҖ мӮ¬мӢӨмғҒ л¶Ҳк°ҖлҠҘн•ҳлӢӨ. - л”°лқјм„ң, мқҙ л¶Җл¶„мқҖ м—¬м „нһҲ мң„нҳ‘мңјлЎң лӮЁм•„ мһҲлӢӨ. ### Construct validity мқҙ м—°кө¬м—җм„ң нҷ•мқён•ң к°Ғ Datasetм—җ лҢҖн•ң provenanceлӮҳ lineageк°Җ м •нҷ•н•ҳм§Җ м•Ҡмқ„ мҲҳ мһҲлӢӨ. - Datasetмқҙ м–ём ң, м–ҙл””м—җм„ң мғқм„ұлҗҳм—ҲлҠ”м§ҖлҘј м •нҷ•нһҲ нҷ•мқён•ҳлҠ” кІғмқҖ л¶Ҳк°ҖлҠҘн•ҳлӢӨ. - ImageNetкіј к°ҷмқҖ кІҪмҡ°, м •нҷ•н•ң Data SourceлҘј м•Ң мҲҳлҸ„ м—ҶлӢӨ. - к°ҖлҠҘн•ң кҙҖл Ё л¬ём„ңлҘј нҶөн•ҙ Data SourceлҘј мң м¶”н•ҳм—¬ лқјмқҙм„ мҠӨлҘј мң м¶”н•  мҲҳ мһҲлҸ„лЎқ мөңм„ мқ„ лӢӨн•  лҝҗмқҙлӢӨ.

мқҙлҹ»кІҢ мң„м—җм„ң м„ӨлӘ…н•ң data pointмқҳ лқјмқҙм„ мҠӨлӮҳ м •нҷ•н•ҳм§Җ м•ҠмқҖ м •ліҙлЎң лқјмқҙм„ мҠӨлҘј нҷ•мқён•  мҲҳ м—ҶлҠ” м–ҙл ӨмӣҖк№Ңм§Җ кі л Өн•ңлӢӨл©ҙ кіөк°ң Datasetмқ„ мғҒмҡ© AI м„ңл№„мҠӨм—җ лқјмқҙм„ мҠӨ лҰ¬мҠӨнҒ¬ м—Ҷмқҙ мӮ¬мҡ©н•ҳлҠ” кІғмқҖ м •л§җ кұ°мқҳ л¶Ҳк°ҖлҠҘн•ҳлӢӨкі  лҙҗм•ј н•ҳлҠ” кІғ м•„лӢҢк°Җ мӢ¶мҠөлӢҲлӢӨ. к·ёл ҮлӢӨкі  AI м ңн’Ҳмқ„ м—°кө¬н•ҳлҠ” лҚ° кіөк°ң Datasetмқ„ м•„мҳҲ л°°м ңн•  мҲҳлҸ„ м—ҶмҠөлӢҲлӢӨ. GitHubк°Җ м Җмһ‘к¶Ң м№Ён•ҙ мқҙмҠҲк°Җ мһҲмқҢм—җлҸ„ л¶Ҳкө¬н•ҳкі  Copilot м„ңл№„мҠӨлҘј мӨҖл№„н•ҳлҠ” кІғмқҖ мқјм • л¶Җл¶„ лІ•м Ғ лҰ¬мҠӨнҒ¬лҘј к°җмҲҳн•ҳкі , н•„мҡ”м—җ л”°лқј лІ•м • лӢӨнҲјлҸ„ мқҙм–ҙк°ҖлҠ” кІғкіј к°ҷмқҙ кё°м—…мқҙ AI кё°мҲ  нҷңмҡ©мқ„ мң„н•ҙ м–ҙлҠҗм •лҸ„мқҳ мһ мһ¬м Ғмқё м Җмһ‘к¶Ң м№Ён•ҙ лҰ¬мҠӨнҒ¬лҠ” л¶ҖлӢҙн•ҳлҠ” кІғлҸ„ кі л Өн•  н•„мҡ”к°Җ мһҲм–ҙ ліҙмһ…лӢҲлӢӨ. мӮ¬мӢӨ, Datasetмқ„ Machine Learning н•ҷмҠөм—җл§Ң мӮ¬мҡ©н•ҳлҠ” кІғмқҖ м Җмһ‘к¶Ң м№Ён•ҙм—җ н•ҙлӢ№н•ҳм§Җ м•ҠлҠ”лӢӨлҠ” кІ¬н•ҙлҸ„ мһҲмҠөлӢҲлӢӨ.

  • м Җмһ‘к¶ҢлІ• м ң35мЎ°мқҳ 2м—җ л”°лҘҙл©ҙ ‘м Җмһ‘л¬јмқ„ к·ё м»ҙн“Ён„°м—җ мқјмӢңм ҒмңјлЎң ліөм ңн•  мҲҳ мһҲлӢӨ’кі  н—Ҳмҡ©н•©лӢҲлӢӨ. мқҙм—җ л”°лқј Machine Learning training кіјм •м—җм„ң кіөк°ң Datasetмқ„ л©”лӘЁлҰ¬м—җ мқјмӢңм ҒмңјлЎң ліөм ңн•ҳлҠ” кІғлҸ„ н—Ҳмҡ©лҗңлӢӨкі  мЈјмһҘн•  м—¬м§Җк°Җ мһҲмҠөлӢҲлӢӨ.
  • м Җмһ‘к¶ҢлІ• м ң35мЎ°мқҳ3м—җм„ңлҠ” м Җмһ‘л¬јмқҳ нҶөмғҒм Ғмқё мқҙмҡ© л°©лІ•кіј м¶©лҸҢн•ҳм§Җ м•„лӢҲн•ҳкі  м Җмһ‘мһҗмқҳ м •лӢ№н•ң мқҙмқөмқ„ л¶ҖлӢ№н•ҳкІҢ н•ҙм№ҳм§Җ м•„лӢҲн•ҳлҠ” кІҪмҡ° кіөм • мқҙмҡ©м—җ н•ҙлӢ№н•ҳм—¬ м Җмһ‘л¬јмқ„ мқҙмҡ©н•  мҲҳ мһҲлӢӨкі  н—Ҳмҡ©н•©лӢҲлӢӨ. мқҙлҜём§Җ м •ліҙлЎң кө¬м„ұлҗң кіөк°ң Datasetмқ„ Machine Learning н•ҷмҠөм—җл§Ң мӮ¬мҡ©н•ҳлҠ” кІғмқҖ к·ёлҰјмқҙлӮҳ мӮ¬м§„мқҳ нҶөмғҒм Ғмқё мқҙмҡ© л°©лІ•кіј м¶©лҸҢн•ҳм§Җ м•Ҡкі , м Җмһ‘мһҗмқҳ мқҙмқөмқ„ н•ҙм№ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ кіөм •мқҙмҡ©м—җ н•ҙлӢ№н•ңлӢӨкі  мЈјмһҘн•  мҲҳ мһҲмқ„ кІғмһ…лӢҲлӢӨ.
